Swapping (hx-swap)

default: content replaces innerHTML of target element.
innerHTML
default: puts content inside target element
outerHTML
replaces entire target lement with the returned content
afterbegin
prepends the content before the first child inside the target
beforebegin
prepends the content before the target in the targets parent element
beforeend
appends the content after the last child inside the target
afterend
appends the content after the target in the targets parent element
delete
delete target element regardless of response
none
don't append content from response (Out of Band Swaps and Response Headers will be processed)
